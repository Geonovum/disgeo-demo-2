This document describes the lessons learned while developing a demonstrator for DisGeo in 5 days. The DisGeo Demo phase 2 shows what Linked Data can mean for DisGeo, and what the shortcomings are. The emphasis is on integral usage questions across datasets, based on coherence between datasets.

The main conclusions are:

  • Linked data is indispensable in realizing the DisGeo ambition to use data in conjunction.
  • Linked data is ready. There are no longer any major technical obstacles. Support in the geographic linked data market is good.
  • Linked data can serve as a direct access layer for users, and can also be the basis for APIs and viewers. This allows different user groups to be served from one source.

Many of the datasets that we use in the demonstrator are published on the basis of linked data standards, but do not contain links to other data, not even to existing key registers. When the links between individual objects from different registrations do exist, working with the data becomes much easier. Supported by the technology of linked data, coherence really works, also between official government registrations and private supplementary registrations. Linked data does not mean per s? linked OPEN data; and closed and open linked data can be linked to each other, whereby authentication and authorization of the closed data with standard web techniques are simple and well arranged. The technology is ready for it; From an organizational point of view, there is still something to be done: establishing links between objects requires governance at the interface, i.e. a responsible party who not only manages the objects but also the links between them.

Software for storing and querying linked data appeared to have good support for geographic linked data. This was not the case a few years ago, but in the meantime a professionalization has taken place in this type of software. We also see the same in other linked data tooling: there is good tooling available for editing, browsing, etc. and this tooling works completely interoperable. This is an important observation, because while data standards have always promised to achieve interoperability, it has never been fully delivered in practice. In this case, yes. Given the good support in the market, we can say that linked data is ready to be used in serious projects with geographic data.
